# Сервис рассылок

Веб-приложение на Django для управления рассылками сообщений клиентам. Проект позволяет создавать, просматривать, 
редактировать и удалять получателей, сообщения и рассылки, 
а также отправлять рассылки вручную через интерфейс и консоль.

## Описание

Сервис предоставляет следующие возможности:

- Управление клиентами (получателями рассылок): добавление, просмотр, редактирование, удаление.
- Управление сообщениями: создание, редактирование, просмотр и удаление шаблонов письма.
- Управление рассылками: создание, просмотр, редактирование и удаление рассылок с выбором получателей и сообщений.
- Отправка рассылок вручную через веб-интерфейс и команду в консоли.
- Регистрация и аутентификация пользователей с разделением ролей (пользователь, менеджер, администратор).
- Ограничение доступа к данным в зависимости от прав пользователя.
- Кеширование для повышения производительности.

## Технологии

- Python 3.11+
- Django 5.2+
- PostgreSQL (в качестве базы данных)
- Redis (для кеширования)
- Bootstrap 5 (для стилей и модальных окон)
- jQuery (для AJAX-запросов)

## Пользователи и права

- **Пользователь**: управляет только своими получателями, сообщениями и рассылками.
- **Менеджер**: может просматривать всех получателей и рассылки, но не может изменять чужие данные.
- **Администратор**: полный доступ ко всем функциям.
